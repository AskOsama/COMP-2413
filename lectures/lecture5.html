1. FROM and WHERE Clauses

SQL Query Syntax

A basic SQL query has the following structure:

SELECT column1, column2, ... FROM table_name WHERE condition;

The WHERE clause supports logical connectives (AND, OR, NOT) and comparison operators (<, <=, >, >=, =, <>).

Example 1: Find the Sid and name of male students who have age > 20

SELECT Sid, Sname FROM Student WHERE sex = 'M' AND age > 20;

SELECT Pid, Pname FROM Professor WHERE grade = 'Assistant Professor';

2. String Operations (LIKE)

Definition: LIKE Operator

SQL includes a string-matching operator for comparisons on character strings. The operator LIKE uses patterns that are described using two special characters:

%

Percent (%)

Matches any substring (including empty string)

_

Underscore (_)

Matches any single character

Example 5: Find the id and names of students whose name includes the substring 'ra'

SELECT Sid, Sname FROM Student WHERE Sname LIKE '%ra%';

Query Result

SidSname
2Sarra Al Askar
3Munera Al Jomoa

Example 6: Find the Cid and the name of all programming courses

SELECT Cid, Cname FROM Course WHERE Cname LIKE '%programming%';

Query Result

CidCname
C1Programming 2
C3Programming1
C5Visual Programming
C6Internet Programming

LIKE Pattern Examples

Pattern Matches
'%ing' Any string ending with "ing"
'Pro%' Any string starting with "Pro"
'%gram%' Any string containing "gram"
'_r%' Any string with "r" as the second character

3. ORDER BY Clause

Definition: ORDER BY

The ORDER BY clause orders the rows in a query result in ascending (ASC) or descending (DESC) order. The default order is ascending.

Syntax

SELECT column1, column2 FROM table_name WHERE condition ORDER BY column_name ASC|DESC;

Example 7: Find the Sid, name and age of male students ordered by descending age

SELECT Sid, Sname, Age FROM Student WHERE sex = 'M' ORDER BY Age DESC;

Query Result

SidSnameAge
4Ali Al Askar25
1Mohamed Al Tamimi22
5Salah Al Ghamdi20

ASC vs DESC

ASC (ascending) sorts from smallest to largest (A-Z, 0-9). DESC (descending) sorts from largest to smallest (Z-A, 9-0). If no order is specified, ASC is the default.

Summary and Key Takeaways

What We've Learned

  1. SQL Query Structure: The basic query uses SELECT ... FROM ... WHERE ... to retrieve filtered data from tables.
  2. WHERE Clause: Filters rows using comparison operators (=, <>, >, <, >=, <=) and logical connectives (AND, OR, NOT).
  3. DISTINCT: Eliminates duplicate rows from query results using SELECT DISTINCT.
  4. String Operations: The LIKE operator with % (any substring) and _ (any single character) enables pattern matching on string columns.
  5. ORDER BY: Sorts query results in ascending (ASC) or descending (DESC) order. Default is ascending.
