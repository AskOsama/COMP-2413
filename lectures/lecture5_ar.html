أهداف التعلم

بنهاية هذه المحاضرة، ستكون قادراً على:

  • كتابة استعلامات SQL باستخدام جمل SELECT و FROM و WHERE
  • استخدام الروابط المنطقية (AND و OR و NOT) في شروط WHERE
  • تطبيق معاملات المقارنة (< و <= و > و >= و = و <>) في الاستعلامات
  • استخدام معامل LIKE مع أحرف البدل (% و _) لمطابقة النصوص
  • استخدام DISTINCT لإزالة الصفوف المكررة من النتائج
  • ترتيب نتائج الاستعلام باستخدام ORDER BY (تصاعدياً ASC وتنازلياً DESC)

1. جملتا FROM و WHERE

صيغة استعلام SQL

الاستعلام الأساسي في SQL له البنية التالية:

SELECT column1, column2, ... FROM table_name WHERE condition;

تدعم جملة WHERE الروابط المنطقية (AND و OR و NOT) ومعاملات المقارنة (< و <= و > و >= و = و <>).

مثال 1: أوجد رقم واسم الطلاب الذكور الذين أعمارهم أكبر من 20

SELECT Sid, Sname FROM Student WHERE sex = 'M' AND age > 20;

نتيجة الاستعلام

SidSname
1Mohamed Al Tamimi
4Ali Al Askar

مثال 2: أوجد رقم واسم جميع الأساتذة المساعدين

SELECT Pid, Pname FROM Professor WHERE grade = 'Assistant Professor';

نتيجة الاستعلام

PidPname
2Dr. Fahd
3Dr. Ameera

مثال 3: أوجد درجة الطالب 3 في المقرر C4

SELECT Mark FROM Marks WHERE Sid = 3 AND Cid = 'C4';

نتيجة الاستعلام

Mark

مثال 4: أوجد رتبة جميع الأساتذة

SELECT Grade FROM Professor;

نتيجة الاستعلام (مع التكرار)

2. العمليات النصية (LIKE)

تعريف: معامل LIKE

تتضمن SQL معامل مطابقة النصوص للمقارنات على السلاسل النصية. يستخدم المعامل LIKE أنماطاً توصف باستخدام حرفين خاصين:

%

علامة النسبة المئوية (%)

تطابق أي سلسلة فرعية (بما في ذلك السلسلة الفارغة)

_

الشرطة السفلية (_)

تطابق أي حرف واحد

مثال 5: أوجد رقم وأسماء الطلاب الذين يحتوي اسمهم على 'ra'

SELECT Sid, Sname FROM Student WHERE Sname LIKE '%ra%';

نتيجة الاستعلام

SidSname
2Sarra Al Askar
3Munera Al Jomoa

مثال 6: أوجد رقم واسم جميع مقررات البرمجة

SELECT Cid, Cname FROM Course WHERE Cname LIKE '%programming%';

نتيجة الاستعلام

CidCname
C1Programming 2
C3Programming1
C5Visual Programming
C6Internet Programming

أمثلة على أنماط LIKE

النمط يطابق
'%ing' أي نص ينتهي بـ "ing"
'Pro%' أي نص يبدأ بـ "Pro"
'%gram%' أي نص يحتوي على "gram"
'_r%' أي نص حرفه الثاني "r"

3. جملة ORDER BY

تعريف: ORDER BY

جملة ORDER BY ترتب الصفوف في نتيجة الاستعلام بترتيب تصاعدي (ASC) أو تنازلي (DESC). الترتيب الافتراضي هو تصاعدي.

الصيغة

SELECT column1, column2 FROM table_name WHERE condition ORDER BY column_name ASC|DESC;

مثال 7: أوجد رقم واسم وعمر الطلاب الذكور مرتبين تنازلياً حسب العمر

SELECT Sid, Sname, Age FROM Student WHERE sex = 'M' ORDER BY Age DESC;

نتيجة الاستعلام

SidSnameAge
4Ali Al Askar25
1Mohamed Al Tamimi22
5Salah Al Ghamdi20

ASC مقابل DESC

ASC (تصاعدي) يرتب من الأصغر إلى الأكبر (A-Z، 0-9). DESC (تنازلي) يرتب من الأكبر إلى الأصغر (Z-A، 9-0). إذا لم يتم تحديد الترتيب، فإن ASC هو الافتراضي.

SELECT Sid, Mark FROM Marks WHERE Cid = 'C3';
الخيارالنتيجة

السؤال 7: أي استعلام SQL يجد درجة الطالب 1 في المقرر 'C2'؟

  1. SELECT Sid, Mark FROM Marks WHERE Cid='C2';
  2. SELECT Mark FROM Marks WHERE Cid='C2' AND Sid <> 1;
  3. SELECT Mark FROM Marks WHERE Cid='C2' OR Sid=1;
  4. SELECT Mark FROM Marks WHERE Cid='C2' AND Sid=1;

السؤال 8: أي استعلام SQL يجد رقم الطالب والدرجات في المقرر 'C4' مرتبة تصاعدياً حسب رقم الطالب؟

  1. SELECT Sid, Mark FROM Marks WHERE Cid='C4' ORDER BY Sid ASC;
  2. SELECT Sid, Mark FROM Marks WHERE Sid=4 ORDER BY Sid DESC;
  3. SELECT Sid, Mark FROM Marks WHERE Cid='C4' ORDER BY Sid DESC;
  4. SELECT Max(Mark) FROM Marks WHERE Cid='C4' ORDER BY Sid ASC;

الملخص والنقاط الرئيسية

ما تعلمناه

  1. بنية استعلام SQL: الاستعلام الأساسي يستخدم SELECT ... FROM ... WHERE ... لاسترجاع البيانات المصفاة من الجداول.
  2. جملة WHERE: تصفي الصفوف باستخدام معاملات المقارنة (= و <> و > و < و >= و <=) والروابط المنطقية (AND و OR و NOT).
  3. DISTINCT: يزيل الصفوف المكررة من نتائج الاستعلام باستخدام SELECT DISTINCT.
  4. العمليات النصية: معامل LIKE مع % (أي سلسلة فرعية) و _ (أي حرف واحد) يمكّن من مطابقة الأنماط على الأعمدة النصية.
  5. ORDER BY: يرتب نتائج الاستعلام بترتيب تصاعدي (ASC) أو تنازلي (DESC). الافتراضي هو تصاعدي.

نظرة مستقبلية: في المحاضرة القادمة، سنستكشف مواضيع متقدمة في استعلامات SQL تشمل دوال التجميع و GROUP BY و HAVING وأنماط استعلامات أكثر تعقيداً.
