برمجة قواعد البيانات
Database Programming

COMP 2413 | د. أسامة المرشد | أساسيات إدارة البيانات

أتقن أساسيات قواعد البيانات، من فهم البيانات والمعلومات إلى تصميم أنظمة قواعد بيانات فعالة باستخدام SQL ومنصات DBMS الحديثة.

نبذة عن المقرر

مرحباً بك في مقرر برمجة قواعد البيانات. يغطي هذا المقرر الشامل المفاهيم والمهارات الأساسية التي يحتاجها كل مبرمج. ستتعلم كيفية تخزين وتنظيم واسترجاع البيانات بكفاءة، وفهم المبادئ التي تشغل أنظمة المعلومات والتطبيقات الحديثة.

ماذا ستتعلم

  • البيانات مقابل المعلومات
  • مفاهيم قواعد البيانات (الكيان، السجل، السمة)
  • أساسيات نظم إدارة قواعد البيانات
  • لغة SQL (DDL، DML، DCL)
  • مبادئ تصميم قواعد البيانات
  • نمذجة البيانات ومخططات ER
  • تقنيات التطبيع
  • تحسين الاستعلامات

لمن هذا المقرر

  • طلاب علوم الحاسب
  • مطوري البرمجيات
  • محللي ومهندسي البيانات
  • محترفي تقنية المعلومات
  • أي شخص يعمل مع أنظمة البيانات

شكل المقرر

  • دروس تفاعلية بتنسيق HTML

لغة تعريف البيانات SQL DDL

تعلم كيفية إنشاء وإدارة هياكل قواعد البيانات باستخدام أوامر DDL بما في ذلك CREATE و ALTER و DROP.

المواضيع المغطاة:

  • CREATE DATABASE/TABLE
  • أنواع البيانات
  • القيود (PRIMARY KEY، FOREIGN KEY)
  • عبارات ALTER و DROP

لغة معالجة البيانات SQL (DML)

تعلم أساسيات أوامر DML لإدراج البيانات في الجداول واسترجاع البيانات باستخدام SELECT مع وبدون شروط.

المواضيع المغطاة:

  • نظرة عامة على DML (INSERT، SELECT، UPDATE، DELETE)
  • عبارات INSERT INTO
  • استعلامات SELECT (جميع الأعمدة وأعمدة محددة)
  • SELECT مع شروط WHERE

حذف وتحديث البيانات ولغة التحكم SQL

أتقن عبارات DELETE و UPDATE لتعديل البيانات. تعلم أوامر DCL (GRANT و REVOKE) للتحكم في صلاحيات الوصول.

المواضيع المغطاة:

  • عبارات DELETE (مشروطة وجميع الصفوف)
  • عبارات UPDATE (جميع الصفوف وصفوف محددة)
  • تحديث أعمدة متعددة
  • لغة التحكم في البيانات (GRANT و REVOKE)

استعلامات SQL الأساسية

تعلم كتابة استعلامات SQL الأساسية باستخدام جمل FROM و WHERE و LIKE و ORDER BY مع مخطط قاعدة بيانات الجامعة.

المواضيع المغطاة:

  • جملتا FROM و WHERE
  • العمليات النصية (LIKE)
  • جملة ORDER BY
  • الكلمة المفتاحية DISTINCT

استعلامات SQL المتقدمة

أتقن الدوال التجميعية والاستعلامات الفرعية المتداخلة والأعمدة المحسوبة لاسترجاع وتحليل البيانات المتقدم في SQL.

المواضيع المغطاة:

  • الدوال التجميعية (COUNT، AVG، MIN، MAX، SUM)
  • GROUP BY و HAVING
  • الاستعلامات الفرعية المتداخلة
  • الأعمدة المحسوبة
